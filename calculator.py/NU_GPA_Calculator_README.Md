# Nigerian University GPA Calculator

A simple Python-based GPA calculator designed using the Nigerian university grading system.

This program allows students to:

* Input course codes
* Enter credit units
* Input grades (A–F)
* Automatically calculate GPA
* View total credit units
* View total quality points
* Determine degree classification

---

## Features

* Nigerian 5-point grading system
* Input validation
* Automatic GPA computation
* Degree classification system
* Clean terminal output formatting
* Beginner-friendly Python project

---

## Nigerian Grading System

| Grade | Point |
| ----- | ----- |
| A     | 5.0   |
| B     | 4.0   |
| C     | 3.0   |
| D     | 2.0   |
| E     | 1.0   |
| F     | 0.0   |

---

## Degree Classification

| GPA Range   | Classification     |
| ----------- | ------------------ |
| 4.50 – 5.00 | First Class        |
| 3.50 – 4.49 | Second Class Upper |
| 2.40 – 3.49 | Second Class Lower |
| 1.50 – 2.39 | Third Class        |
| 1.00 – 1.49 | Pass               |
| Below 1.00  | Fail               |
