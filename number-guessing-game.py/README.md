# NUMBER GUESSING GAME

## A python program that allows users to guess an assigned value(integer) in a given range(0 and 20).

# FEATURES;
- An integer is preassigned as the “secret_number”.
- It accepts only integer input from the user.
- A while loop allows the user to play until the secret_number is guessed correctly.
- A counter  was added to give the number of attempts made till correct guess.
