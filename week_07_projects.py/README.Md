# Python Mini Projects Collection

A collection of beginner-friendly Python projects developed to strengthen my understanding of programming fundamentals, problem-solving, functions, loops, conditional statements, data structures, and user interaction.

---

## Projects Included

### 1. Expense Tracker

A simple command-line application that helps users track their expenses and monitor spending habits.

#### Features
- Add new expenses
- Categorize expenses
- Calculate total expenses
- View expense records
- Save and manage expense data
- Simple and user-friendly interface

#### Concepts Used
- Variables
- Lists
- Functions
- Loops
- Conditional Statements
- User Input Handling

---

### 2. Department Inventory Management System

A command-line inventory management system designed to manage department equipment and resources.

#### Features
- View available items
- Borrow equipment
- Return borrowed equipment
- Add new inventory items
- Automatic status updates
- Search available equipment using list comprehensions

#### Concepts Used
- Lists and Nested Lists
- Functions
- Loops
- Conditional Statements
- List Comprehensions
- Data Management

---

### 3. Logic Gate Simulator

A Python program that simulates the behavior of basic digital logic gates.

#### Supported Gates
- AND Gate
- OR Gate
- NOT Gate
- XOR Gate

#### Features
- Accepts binary inputs (0 and 1)
- Simulates real logic gate operations
- Displays gate outputs
- Half Adder
- Input validation for accurate results
- Interactive menu system

#### Concepts Used
- Boolean Logic
- Functions
- Conditional Statements
- User Input Validation
- Program Flow Control
